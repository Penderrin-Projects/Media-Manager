Media Manager

Server settings — changes are saved to config.json on the NAS
qBittorrent (Seedbox)
Your seedbox's qBittorrent web interface URL. Find it in your seedbox provider's dashboard.
Your qBittorrent login credentials — usually provided by your seedbox host.
SFTP (Seedbox File Transfer)
Used to transfer completed downloads from your seedbox to the NAS. These credentials come from your seedbox provider.
The folder on your seedbox where qBittorrent saves completed downloads.
Prowlarr (Indexer Search)
Your Prowlarr instance URL. Install Prowlarr from prowlarr.com — it manages torrent indexer searches.
In Prowlarr: Settings → General → Security → API Key. Copy and paste it here.
TMDB (Metadata & Renaming)
Free API key for movie/TV metadata and posters. Get one in 2 minutes: 1. Create account → 2. Request API key → Copy the "API Key (v3 auth)" value.
Plex (Duplicate Detection) — Optional
Your local Plex server URL. Usually http://YOUR_NAS_IP:32400
Warns before grabbing content already in Plex. To find your token: open Plex Web → pick any media → Get Info → View XML → copy the "X-Plex-Token" from the URL bar.
Media Libraries
Define where your media goes. Paths are container-internal — they map to your NAS folders via Docker volume mounts. Movies and TV Shows are required. Add more for additional libraries (anime, kids, etc).
Temporary workspace for downloads before renaming. Gets cleaned up after each job.
Library Name Container Path Shows?
New libraries default to movie handling. Check "Shows?" for TV show logic (episode matching, season folders).
Direct to PC
Where files go when you use "To My PC". Uses the PC path format (e.g. C:\Users\tabor\Desktop\Torrents).
Server
Default: 9876. Change only if you have a conflict.
Protects the API. Leave blank if only accessible on your local network.
SMS Notifications (Gmail SMTP) — Optional
Get notified via text when downloads are ready in Plex. Requires a Gmail account with an app password. Generate an app password here (you may need to enable 2-Step Verification first).
